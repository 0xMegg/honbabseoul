# 혼밥서울 (ホンバプソウル) MVP v1.0 — 기능 정의서

> Last updated: 2026-04-24
> Source: 사용자 전달 기획서 원본 (MVP v1.0)

## 1. 프로젝트 개요

- **서비스명:** 혼밥서울 (ホンバプソウル — おひとりさま専用・ソウルグルメマップ)
- **타겟 유저:** 한국을 방문하는 일본인 2030 여성 나홀로 여행객
- **핵심 가치:** 눈치 보지 않고 1인 식사가 가능한 서울 식당 큐레이션 및 지도 제공

### 기술 스택 (제안)
- **Frontend:** Next.js
- **Hosting / Deploy:** Vercel (PaaS)
- **Database / Admin:** Supabase
- **Map API:** Naver Maps API

---

## 2. 공통 요건 (UI/UX)

- **언어 설정:** 서비스 내 모든 UI 텍스트(메뉴, 버튼, 필터, 안내 문구)는 일본어를 기본으로 함
- **브랜딩 (로고):** 로고 타이포그래피는 한글 **'혼밥서울'**을 메인 그래픽으로 사용하고, 서브 카피로 카타카나를 작게 배치하여 '찐 한국 로컬 서비스'의 힙한 감성을 강조
- **반응형 웹:** 타겟 특성상 모바일 접속이 압도적 → **Mobile-first** 웹뷰 사이즈에 최적화

---

## 3. 핵심 필터 로직 (중요)

필터는 유저 편의를 위해 **'포지티브 필터(Positive Filter)'** 방식을 채택.

### [기본값] 혼밥 가능 (一人食いOK)
- 서비스 접속 시 **무조건 기본으로 활성화**되어 있어야 함
- **활성 시:** 1인 식사가 가능한 식당만 지도에 노출
- **비활성 시(필터 해제 시):** 2인 이상만 가능한 식당(무한리필, 전골 등)까지 모두 노출

### [추가 필터 1] 일본어 메뉴 제공 (日本語メニューあり)
일본어 메뉴판이 있거나 일본어 응대가 가능한 식당

### [추가 필터 2] 심야 영업 (深夜営業)
밤 10시(22:00) 이후에도 영업을 지속하는 식당

---

## 4. 사용자 화면 구성 (Frontend)

### 4.1. 메인 지도 (Naver Maps API)
- 접속 시 내 위치 기반 또는 지정된 메인 상권 중심으로 지도 렌더링
- DB에 등록된 식당 위치에 핀(마커) 노출

### 4.2. 식당 상세 정보 (바텀 시트)
- 핀 클릭 시 화면 하단에서 **바텀 시트** 형태로 상세 정보 노출
- **시각 자료 강조:** 대표 음식 및 메뉴판 사진 필수 노출
- **상세 데이터:** 식당명(일문/국문 병기), 가격대, 상세 주소(복사 가능)
- **외부 링크:** "네이버 지도로 보기" 버튼 클릭 시, 앱 설치 유도가 아닌 **네이버 지도 웹 브라우저(Web)**로 즉시 연결되도록 처리

---

## 5. 사용자 제보 기능 (UGC - User Generated Content)

사용자가 직접 맛집을 제보할 수 있는 폼 제공. 데이터 퀄리티 유지를 위해 입력 조건을 엄격하게 제한.

### 입력 폼 구성 (사진 제외 ALL 필수값)
- **[필수]** 식당 이름
- **[필수]** 네이버 지도 링크 (장소 확인용)
- **[필수]** 혼밥 가능 여부 (Yes/No)
- **[필수]** 일본어 메뉴 제공 여부 (Yes/No)
- **[필수]** 밤 10시 이후 영업 여부 (Yes/No)
- **[필수]** 추천 이유 (현지인 팁 등 텍스트)
- **[선택]** 음식 또는 메뉴판 사진 업로드

### 유효성 검사
- 위 필수 항목이 하나라도 누락되면 '제출' 버튼 비활성화

### 검수 프로세스
- 제출된 데이터는 **즉시 지도에 반영되지 않고**, DB에 `Pending`(승인 대기) 상태로 저장
- 관리자가 내용 확인 후 `Approved`(승인) 처리 시 지도에 라이브 노출

---

## 6. 데이터베이스 구조 (Supabase)

| 필드 | 설명 |
|---|---|
| `status` | 제보 데이터 검수 상태 (`pending` / `approved` / `rejected`) |
| `is_solo_default` | 혼밥 가능 여부 (Boolean, Default: `true`) |
| `has_jp_menu` | 일본어 메뉴 유무 (Boolean) |
| `is_late_night` | 22시 이후 심야 영업 여부 (Boolean) |

관리자(디렉터)가 **Supabase 대시보드에서 직접 엑셀처럼 데이터를 검수/추가/수정**할 수 있도록 권한 및 테이블 세팅 필수.
